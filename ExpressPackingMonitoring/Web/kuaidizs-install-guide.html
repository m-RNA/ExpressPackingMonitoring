订单备注播报插件安装向导

按下面步骤操作。安装完成后，快递助手里的买家留言、卖家备注会发送到监控工位，打包时自动播报。

1

安装 Tampermonkey / 篡改猴

先给浏览器安装脚本管理扩展。如果已经安装过，可以跳过这一步。

Chrome 安装 Edge 安装 其他浏览器
2

打开浏览器的用户脚本权限

Chrome / Edge 新版本需要在扩展详情里打开“允许用户脚本”；如果没有这个开关，就打开“开发者模式”。本地安装脚本时，也建议打开“允许访问文件网址”。

  • 打开浏览器扩展管理页面，找到 Tampermonkey / 篡改猴。
  • 进入“详情”，打开“允许用户脚本”。
  • 如果没有“允许用户脚本”，打开“开发者模式”：Edge 一般在左下角，Chrome 一般在右上角。
  • 如果有“允许访问文件网址”，也一起打开。
查看图文说明
复制后粘贴到浏览器地址栏打开。浏览器通常会拦截网页直接打开 chrome:// 或 edge:// 地址。
3

安装或更新订单备注插件脚本

请点击下面按钮，让 Tampermonkey 打开脚本安装页；以前安装过也要点击“更新”或“覆盖安装”。不要用记事本、Visual Studio 或脚本宿主打开这个文件。

{{scriptLink}}
从当前监控工位的安装向导更新脚本，会自动加入该工位地址的精确跨源访问权限，可避免反复出现“跨源资源请求”确认框。如果更新前仍出现确认框，请选择“始终允许此域名”（当前监控工位 IP），不要选择“始终允许所有域名”。如果浏览器只显示代码，请确认第 1 步已安装 Tampermonkey，并完成第 2 步的权限开关。
从当前电脑更新脚本时，已连接电脑备份的手机会自动加入订单接收列表，订单会同时发送到在线设备
4

填写摄像头监控工位地址

程序已尝试把这个地址复制到剪贴板。脚本里要求填写地址时，直接粘贴即可。

{{address}}
如果这里为空，请先回到打印工位窗口连接摄像头监控工位。
5

发送测试订单

安装后打开 Tampermonkey 菜单，点击“发送测试订单”，用于确认监控工位能否收到订单备注。

监控工位收到后会播放提示音并播报“收到测试订单”。
6

开启扫码退款核验

保持一个已登录的快递助手批量打印页面打开。插件会在后台打开一个不抢焦点的“退款核验工作页”，只有该专用页面会切换官方“打印后退款”筛选，不会改变用户正在操作的打印页面。普通订单仍由用户的打印页面自动推送，不依赖专用工作页。

只有确认监控端在线时才会自动创建专用工作页。专用工作页带有明显标题和半透明遮罩，请勿手动操作；误关后插件会自动重建，也可从扩展菜单手动打开。监控端扫码后，工作页会先回传当前退款列表；目标单号不在列表中时，再按快递单号精确查询历史订单。查询失败时会使用监控端最近 90 天的订单缓存核验。发货或退货模式扫到退款订单时会继续录像，并按退款状态连续报警。
